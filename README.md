# Event Lucky Draw Picker

브라우저에서 바로 실행할 수 있는 오픈소스 경품 추첨 웹앱입니다.  
원본 기능을 유지하면서 GitHub Pages에 바로 올릴 수 있게 정리한 배포용 패키지입니다.

## Features

- 참가자 목록 기반 추첨
- 당첨자 이름 확대 애니메이션
- 축하 멜로디와 박수 효과
- 화면 전체 폭죽 연출
- 전체화면 모드 지원
- 당첨 이력 표시

## Quick Start

1. 이 저장소를 다운로드하거나 복제합니다.
2. `index.html`을 브라우저에서 엽니다.
3. GitHub Pages에 배포하려면 저장소 루트에 `index.html`을 둔 뒤 Pages를 켭니다.

## GitHub Pages

- Repository name: `event-lucky-draw`
- Branch: `main`
- Folder: `/(root)`

배포 주소 예시:

`https://YOUR_GITHUB_ID.github.io/event-lucky-draw/`

## Customize

- 상단 행사명은 화면의 입력 필드에서 바로 수정할 수 있습니다.
- 참가자 목록은 줄바꿈 단위로 입력합니다.
- 색상, 효과, 사운드는 `index.html` 안에서 직접 수정할 수 있습니다.

## Files
